== Description ==
Product stock count progress bar for WooCommerce helps to increase your product sales by encouraging your visitors to buy before sold out.

Product stock count progress bar for WooCommerce is a simple lightweight plugin that will show a stock counts with a progress bar for the specific product.

== Installation ==
1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page.
3. OR, Got to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'Product stock count progress bar for WooCommerce'.
4. Install and activate the plugin.
